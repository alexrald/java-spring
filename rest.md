### REST PRINCIPLES
- based on JSON payload
- using HTTP protocol

### HTTP codes family
- 1xx - processing in progress
- 2xx - success
- 3xx - redirects
- 4xx - client error
- 5xx - server error

### HTTP exchange message consists of:
- request
  - url
  - headers
  - body
- response
  - status code
  - headers
  - body
